1. 메뉴 설계 및 스토리보드 작성

  • 무엇을 보여주고자 할지 concept를 구상하고 메인 화면 디자인과 메인 화면에 보여질 메뉴 설계.
  • 전체 메뉴 구성도를 설계하고 각 서브 페이지별 디자인 및 내용 설계.

2. 메인 및 서브페이지 디자인

  • PC, Tablet, Mobile 별 화면 디자인

3. html 작성을 위한 Markup 설계

  • html 구현 전에 전체적인 구도와 구조 설계 및 class 지정 등의 Markup 설계.

4. Markup 제작

  • Editplus, Atom 툴 사용.
  • html, css, jQuery를 사용하여 화면 구현 및 반응형 제작.

5. 웹표준 점검

  • validator.w3.org,
    jigsaw.w3.org/css-validator를 사용하여 html과 css의 웹표준 점검

7. 웹접근성 점검

  • k-wah4.0, OpenWAX를 사용하여 웹접근성 점검
